Getting started

Install StatusForge, sign in with Twitch or Kick, and the presence engine starts scanning running games automatically. No manual setup — overlays generate themselves from what you're already playing.

Drop the generated overlay URL into your broadcast software as a browser source, sized to 1920×280, and it updates live as your game or activity changes.

Configuring overlays

Each overlay reads from the same presence state, so switching games updates every connected output at once. You can restyle position, scale, and which fields show per overlay.

{"game": "Hollow Knight", "status": "Playing", "since": "2026-07-04T18:22:00Z"}

Connecting platforms

Twitch, Kick, and Joystick.tv connect via OAuth from Settings → Integrations. Streamer.bot can subscribe as a relay for chat-triggered actions tied to presence changes.
